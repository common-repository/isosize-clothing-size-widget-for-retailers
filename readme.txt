== Description ==

= Isosize clothing size plugin for wordpress retailers. =

Isosize plugin helps your customers to find the perfect fit by calculating the right clothing size from a huge verity of stores, brands or countries. Ideal for fashion bloggers, webstore integrators and other creative people.
The consumer input the basic 3 measurements (bust girth, waist girth, hips girth ) in cm or inches and isosize calculates the perfect size to select for your own webstore.

Should your customers buy on your behalf or for someone else, isosize also allows to send customers own size to relatives for special occasions such as birthday or Valentine.

This FREE version includes the standard EU Sizing chart only.
You may purchase your own sizing chart on http://www.isosize.com/seller.aspx

Fully compatible with WooCommerce E-Commerce plugin 2.5.5
Fully compatible with Shopify E-Commerce plugin 1.0.2



== Installation ==

= how to install the plugin and get it working : =

 From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'isosize'
3. Activate isosize from your Plugins page. (Make sure it has completed successfully.)

= From WordPress.org =

1. Download isosize clothing size plugin for retailers (isosize-widget).
2. Upload the 'isosize' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate isosize from your Plugins page. (Make sure it has completed successfully.)
